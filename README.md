# Simple Interest Calculator

A bash script that calculates simple interest given principal, annual rate of interest and time period in years.

## Features

* **Principal Input:** Allows users to enter the initial amount of money.
* **Rate Calculation:** Accepts annual interest rate as a percentage.
* **Time Period:** Users can specify the duration in years.
* **Automatic Computation:** Instantly calculates the total simple interest using the formula: `Interest = (P * R * T) / 100`.

## Examples of Usage

### Example 1: Basic Calculation
* **Input:**
    * Principal: 1000
    * Rate: 5%
    * Time: 2 years
* **Output:**
    * Simple Interest: 100

### Example 2: Short-term Investment
* **Input:**
    * Principal: 5000
    * Rate: 7%
    * Time: 1 year
* **Output:**
    * Simple Interest: 350

## How to run
1. Clone the repository.
2. Run `chmod +x simple-interest.sh` to make it executable.
3. Execute with `./simple-interest.sh`.
